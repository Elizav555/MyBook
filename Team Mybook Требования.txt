Стек:
* ASP .NET MVC
* Bootstrap v5
* Google Books API (будем получать инфу о книгах)
* Google Maps API (чтобы на карте магазины отмечать и т.д)
* Auth API (авторизация через вк)
* SignalR (уведомления со стороны сервера, например когда тебе другой человек дарит подписку. Уведомление об истечении подписки. Любимый автор и любимый жанр для юзера и уведомлять о новых книгах этого жанра и этого автора)
* JQuery
